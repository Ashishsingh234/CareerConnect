## ✨ Features

- 🔐 **User Authentication** — JWT-based login & registration
- 💼 **Job Listings** — Browse 10,000+ verified jobs
- 🔍 **Smart Search** — Filter by title, company, or skill
- 📝 **Job Applications** — Apply directly through the platform
- 👤 **User Profiles** — Manage your profile and applications
- 📧 **Email Notifications** — Get notified on application updates
- 📱 **Responsive Design** — Works on all devices

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | React.js + Vite |
| **Backend** | Node.js + Express.js |
| **Database** | MongoDB Atlas |
| **Auth** | JWT (JSON Web Tokens) |
| **Hosting** | Vercel (Frontend) + Render (Backend) |
| **Styling** | CSS3 + Custom Design System |
